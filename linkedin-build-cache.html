CI/CD Reliability Philosophy: Cache as Infrastructure

Core Build Cache Principles

  • Content-Addressable Storage (CAS) — Each artifact identified by SHA-256 hash for integrity
  • Two-Tier Cache Pattern — SSD-based L1 hot cache + Object storage L2 cold retrieval
  • Lease-Based Eviction — Prevents artifact removal while active builds reference it
  • Async Replication + Bloom Filters — Avoids rebuilds during global sync lag
  • Checksum Validation Pipeline — CRC + periodic audit to detect bit-rot
  • Read-Your-Writes Consistency — Guarantee within region, eventual global consistency

Six Core Components

  1. Artifact Ingestion API — Receives newly built artifacts + metadata hash
  2. Hash Index Service — Deduplicates artifacts by content hash (SHA-256)
  3. Storage Layer — Regional object stores (Ambry buckets with CAS addressing)
  4. Cache Layer — Tier-1 hot SSD nodes, Tier-2 cold archive
  5. Replication Agent — Syncs indexes + metadata across regions
  6. CI Client SDK — Queries hash before build, fetches if available

Reliability Stressors (What Can Break)

Critical Failure Modes

  • Stale/Wrong Artifacts: Bad hash computation → non-deterministic builds
  • Eviction Race Condition: Artifact evicted mid-use → pipeline failure
  • Replication Lag: Artifacts missing in remote region → rebuild storm
  • Checksum Drift: Silent corruption in storage tier
  • Cache Pollution: Temporary build artifacts flooding storage
  • Hot Shard Overload: Popular artifacts causing cache hotspots

Content-Addressable Storage (CAS) Implementation

SHA-256 Hash-Based Addressing

Integrity by Design: Every artifact addressable by its content hash

// LinkedIn Build Cache with content addressing
public class ContentAddressableCache {
    
    // Generate content-based address for any artifact
    public String generateContentAddress(ArtifactMetadata artifact, byte[] content) {
        MessageDigest sha256 = MessageDigest.getInstance("SHA-256");
        
        // Include metadata in hash for determinism
        sha256.update(artifact.getBuildInputs().getBytes());
        sha256.update(artifact.getToolchain().getBytes());
        sha256.update(artifact.getCompilerFlags().getBytes());
        sha256.update(content);
        
        byte[] hash = sha256.digest();
        return "sha256:" + Hex.encodeHexString(hash);
    }
    
    // Store artifact with CAS addressing
    public void storeArtifact(BuildArtifact artifact) {
        String contentAddress = generateContentAddress(artifact.getMetadata(), artifact.getContent());
        
        // Check if already exists (deduplication)
        if (hashIndex.contains(contentAddress)) {
            logger.info("Artifact already cached: {}", contentAddress);
            return;
        }
        
        // Store in both hot and cold tiers
        hotCache.put(contentAddress, artifact.getContent());
        coldStorage.put(contentAddress, artifact);
        
        // Update index with metadata
        hashIndex.put(contentAddress, ArtifactIndex.builder()
            .contentAddress(contentAddress)
            .size(artifact.getContent().length)
            .buildTimestamp(Instant.now())
            .accessCount(0)
            .leaseExpiresAt(null)
            .build());
        
        logger.info("Stored artifact: {} size: {} bytes", contentAddress, artifact.getContent().length);
    }
    
    // Retrieve artifact by content hash
    public Optional<BuildArtifact> retrieveArtifact(String contentAddress) {
        ArtifactIndex index = hashIndex.get(contentAddress);
        if (index == null) {
            return Optional.empty();
        }
        
        // Try hot cache first (L1)
        byte[] content = hotCache.get(contentAddress);
        if (content != null) {
            metricsClient.increment("cache.hit.hot");
            return Optional.of(new BuildArtifact(index.getMetadata(), content));
        }
        
        // Fallback to cold storage (L2)
        BuildArtifact artifact = coldStorage.get(contentAddress);
        if (artifact != null) {
            // Promote to hot cache for future access
            hotCache.put(contentAddress, artifact.getContent());
            metricsClient.increment("cache.hit.cold");
            return Optional.of(artifact);
        }
        
        metricsClient.increment("cache.miss");
        return Optional.empty();
    }
}

Two-Tier Cache Implementation

Hot SSD + Cold Object Storage Pattern

// Intelligent cache tier management
public class TwoTierCacheManager {
    
    private final SSDCache hotTier;     // Fast SSD storage (limited capacity)
    private final AmbryCold coldTier;   // LinkedIn Ambry object store (unlimited)
    
    // Hot tier: Recent and frequently accessed artifacts
    public class SSDCache {
        private final Map<String, CacheEntry> cache = new ConcurrentHashMap<>();
        private final long maxSizeBytes = 1_000_000_000_000L; // 1TB SSD
        
        // LRU eviction with lease protection
        public void evictIfNeeded() {
            long currentSize = calculateTotalSize();
            
            if (currentSize > maxSizeBytes * 0.9) { // 90% threshold
                List<CacheEntry> candidates = cache.values().stream()
                    .filter(entry -> !entry.hasActiveLease()) // Never evict leased artifacts
                    .sorted(Comparator.comparing(CacheEntry::getLastAccessTime))
                    .limit(100) // Evict in batches
                    .collect(Collectors.toList());
                
                for (CacheEntry entry : candidates) {
                    cache.remove(entry.getContentAddress());
                    logger.info("Evicted from hot cache: {}", entry.getContentAddress());
                }
            }
        }
    }
    
    // Cold tier: All artifacts with slower access
    public class AmbryColdStorage {
        private final AmbryClient ambryClient;
        
        public void put(String contentAddress, BuildArtifact artifact) {
            // Store in LinkedIn Ambry with CAS key
            BlobId blobId = ambryClient.put(
                contentAddress,
                artifact.serialize(),
                ArtifactTTL.DAYS_90 // 90-day retention
            );
            
            // Update index mapping
            casIndex.put(contentAddress, blobId);
        }
        
        public BuildArtifact get(String contentAddress) {
            BlobId blobId = casIndex.get(contentAddress);
            if (blobId == null) return null;
            
            byte[] data = ambryClient.get(blobId);
            return BuildArtifact.deserialize(data);
        }
    }
    
    // Retrieve with tier promotion
    public Optional<BuildArtifact> get(String contentAddress) {
        // Try hot tier first
        Optional<BuildArtifact> hotResult = hotTier.get(contentAddress);
        if (hotResult.isPresent()) {
            return hotResult;
        }
        
        // Fetch from cold tier and promote
        Optional<BuildArtifact> coldResult = coldTier.get(contentAddress);
        if (coldResult.isPresent()) {
            hotTier.put(contentAddress, coldResult.get());
            metricsClient.increment("cache.promotion");
        }
        
        return coldResult;
    }
}

Lease-Based Eviction Prevention

Active Build Protection

Prevents artifacts from being evicted while builds are actively using them

// Lease system to prevent eviction races
public class LeaseBasedEvictionController {
    
    private final Map<String, Set<LeaseToken>> activeLeases = new ConcurrentHashMap<>();
    
    // Acquire lease before using artifact in build
    public LeaseToken acquireLease(String contentAddress, String buildId, Duration leaseTTL) {
        LeaseToken lease = LeaseToken.builder()
            .contentAddress(contentAddress)
            .buildId(buildId)
            .acquiredAt(Instant.now())
            .expiresAt(Instant.now().plus(leaseTTL))
            .build();
        
        activeLeases.computeIfAbsent(contentAddress, k -> ConcurrentHashMap.newKeySet()).add(lease);
        
        logger.info("Acquired lease for artifact {} build {}", contentAddress, buildId);
        return lease;
    }
    
    // Release lease when build completes
    public void releaseLease(LeaseToken lease) {
        Set<LeaseToken> leases = activeLeases.get(lease.getContentAddress());
        if (leases != null) {
            leases.remove(lease);
            logger.info("Released lease for artifact {} build {}", 
                       lease.getContentAddress(), lease.getBuildId());
        }
    }
    
    // Check if artifact can be safely evicted
    public boolean canEvictArtifact(String contentAddress) {
        Set<LeaseToken> leases = activeLeases.get(contentAddress);
        if (leases == null || leases.isEmpty()) {
            return true;
        }
        
        // Check for expired leases
        Instant now = Instant.now();
        leases.removeIf(lease -> lease.getExpiresAt().isBefore(now));
        
        return leases.isEmpty();
    }
    
    // Periodic cleanup of expired leases
    @Scheduled(fixedDelay = 60000) // every minute
    public void cleanupExpiredLeases() {
        Instant now = Instant.now();
        int cleanedUp = 0;
        
        for (Map.Entry<String, Set<LeaseToken>> entry : activeLeases.entrySet()) {
            Set<LeaseToken> leases = entry.getValue();
            int sizeBefore = leases.size();
            
            leases.removeIf(lease -> lease.getExpiresAt().isBefore(now));
            
            cleanedUp += sizeBefore - leases.size();
            
            // Remove empty lease sets
            if (leases.isEmpty()) {
                activeLeases.remove(entry.getKey());
            }
        }
        
        if (cleanedUp > 0) {
            logger.info("Cleaned up {} expired leases", cleanedUp);
        }
    }
}

Cross-Region Replication with Bloom Filters

Intelligent Replication Strategy

// Async replication with Bloom filter optimization
public class CrossRegionReplicationAgent {
    
    private final BloomFilter<String> replicatedHashes;
    private final Map<String, RegionalCache> regionalCaches;
    
    // Replicate new artifacts across regions
    public void replicateArtifact(String contentAddress, BuildArtifact artifact) {
        CompletableFuture<Void>[] replicationTasks = regionalCaches.entrySet().stream()
            .filter(entry -> !entry.getKey().equals(getCurrentRegion()))
            .map(entry -> replicateToRegionAsync(entry.getValue(), contentAddress, artifact))
            .toArray(CompletableFuture[]::new);
        
        // Fire and forget - don't block builds on replication
        CompletableFuture.allOf(replicationTasks)
            .thenRun(() -> {
                replicatedHashes.put(contentAddress);
                logger.info("Artifact replicated globally: {}", contentAddress);
            })
            .exceptionally(throwable -> {
                logger.error("Replication failed for {}", contentAddress, throwable);
                return null;
            });
    }
    
    private CompletableFuture<Void> replicateToRegionAsync(RegionalCache targetCache, 
                                                          String contentAddress, 
                                                          BuildArtifact artifact) {
        return CompletableFuture.runAsync(() -> {
            // Check if target region already has this artifact
            if (targetCache.contains(contentAddress)) {
                return;
            }
            
            // Replicate both index and data
            targetCache.storeArtifact(contentAddress, artifact);
            
            metricsClient.increment("replication.success", 
                                  Map.of("target_region", targetCache.getRegion()));
        });
    }
    
    // Bloom filter check to avoid unnecessary replication queries
    public boolean isLikelyReplicated(String contentAddress) {
        return replicatedHashes.mightContain(contentAddress);
    }
    
    // Periodic consistency check across regions
    @Scheduled(fixedDelay = 300000) // every 5 minutes
    public void performConsistencyCheck() {
        String primaryRegion = "ltx1";
        RegionalCache primaryCache = regionalCaches.get(primaryRegion);
        
        for (Map.Entry<String, RegionalCache> entry : regionalCaches.entrySet()) {
            if (entry.getKey().equals(primaryRegion)) continue;
            
            RegionalCache secondaryCache = entry.getValue();
            ConsistencyReport report = compareRegionalCaches(primaryCache, secondaryCache);
            
            if (report.getInconsistentArtifacts() > 10) {
                alertManager.warn("High inconsistency between {} and {}: {} artifacts", 
                                primaryRegion, entry.getKey(), report.getInconsistentArtifacts());
            }
            
            // Auto-heal missing artifacts
            healMissingArtifacts(primaryCache, secondaryCache, report.getMissingArtifacts());
        }
    }
}

Checksum Validation & Corruption Detection

Data Integrity Protection

Comprehensive validation to detect and recover from silent corruption

// Multi-layer integrity validation
public class IntegrityValidationPipeline {
    
    // Validate artifact on retrieval
    public ValidationResult validateArtifact(String contentAddress, byte[] content) {
        ValidationResult result = new ValidationResult();
        
        // 1. SHA-256 content hash validation
        String computedHash = "sha256:" + Hex.encodeHexString(
            MessageDigest.getInstance("SHA-256").digest(content)
        );
        
        if (!computedHash.equals(contentAddress)) {
            result.addError("Content hash mismatch: expected " + contentAddress + 
                          " got " + computedHash);
        }
        
        // 2. CRC32 checksum for fast corruption detection
        CRC32 crc = new CRC32();
        crc.update(content);
        long computedCRC = crc.getValue();
        
        ArtifactMetadata metadata = getMetadata(contentAddress);
        if (metadata.getStoredCRC() != computedCRC) {
            result.addError("CRC mismatch: expected " + metadata.getStoredCRC() + 
                          " got " + computedCRC);
        }
        
        // 3. Artifact-specific validation
        if (contentAddress.endsWith(".jar")) {
            result.merge(validateJarFile(content));
        } else if (contentAddress.contains("docker-image")) {
            result.merge(validateDockerImage(content));
        }
        
        return result;
    }
    
    // Periodic background integrity audit
    @Scheduled(fixedDelay = 3600000) // every hour
    public void performIntegrityAudit() {
        // Sample 1% of cached artifacts for validation
        List<String> sampleArtifacts = hashIndex.keySet().stream()
            .filter(hash -> hash.hashCode() % 100 == 0) // 1% sample
            .limit(1000) // Max 1000 per audit
            .collect(Collectors.toList());
        
        int corruptionCount = 0;
        
        for (String contentAddress : sampleArtifacts) {
            try {
                Optional<BuildArtifact> artifact = retrieveArtifact(contentAddress);
                if (artifact.isPresent()) {
                    ValidationResult result = validateArtifact(contentAddress, 
                                                             artifact.get().getContent());
                    if (!result.isValid()) {
                        handleCorruption(contentAddress, result);
                        corruptionCount++;
                    }
                }
            } catch (Exception e) {
                logger.error("Validation failed for {}", contentAddress, e);
            }
        }
        
        double corruptionRate = (double) corruptionCount / sampleArtifacts.size();
        metricsClient.gauge("cache.corruption_rate", corruptionRate);
        
        if (corruptionRate > 0.001) { // 0.1% threshold
            alertManager.page("Cache corruption rate exceeded threshold: {}%", 
                            corruptionRate * 100);
        }
    }
    
    // Handle detected corruption
    private void handleCorruption(String contentAddress, ValidationResult result) {
        logger.error("Corruption detected for {}: {}", contentAddress, result.getErrors());
        
        // 1. Remove corrupted artifact from all caches
        hotCache.remove(contentAddress);
        coldStorage.remove(contentAddress);
        
        // 2. Try to recover from other regions
        Optional<BuildArtifact> recovered = recoverFromOtherRegions(contentAddress);
        if (recovered.isPresent()) {
            storeArtifact(recovered.get());
            logger.info("Recovered corrupted artifact from remote region: {}", contentAddress);
        }
        
        // 3. Alert and mark for rebuild
        alertManager.warn("Artifact corruption detected and removed: {}", contentAddress);
        markForRebuild(contentAddress);
    }
}

Cache Hit Rate Optimization

Intelligent Cache Management

// Hit rate optimization and monitoring
public class CacheOptimizationService {
    
    // Monitor and optimize cache hit rates
    @Scheduled(fixedDelay = 300000) // every 5 minutes
    public void optimizeCachePerformance() {
        CacheMetrics metrics = gatherCacheMetrics();
        
        // Global hit rate tracking
        double globalHitRate = metrics.getHitRate();
        metricsClient.gauge("cache.hit_rate", globalHitRate);
        
        if (globalHitRate < 0.8) { // Below 80%
            alertManager.warn("Cache hit rate below target: {}%", globalHitRate * 100);
            
            // Analyze and take corrective action
            analyzeCacheMisses(metrics);
        }
        
        // Per-artifact-type hit rates
        Map<String, Double> hitRateByType = metrics.getHitRateByArtifactType();
        for (Map.Entry<String, Double> entry : hitRateByType.entrySet()) {
            metricsClient.gauge("cache.hit_rate_by_type", entry.getValue(),
                              Map.of("artifact_type", entry.getKey()));
        }
        
        // Adaptive cache sizing based on hit rates
        adaptCacheSizing(metrics);
    }
    
    private void analyzeCacheMisses(CacheMetrics metrics) {
        // Identify patterns in cache misses
        Map<String, Integer> missByReason = metrics.getMissByReason();
        
        if (missByReason.get("EVICTED_TOO_EARLY") > 100) {
            // Increase hot cache size
            scalingController.scaleHotCache(1.2); // 20% increase
            logger.info("Scaled up hot cache due to premature evictions");
        }
        
        if (missByReason.get("NOT_REPLICATED") > 50) {
            // Improve replication speed
            replicationAgent.increaseReplicationWorkers();
            logger.info("Increased replication workers due to replication lag");
        }
        
        if (missByReason.get("FIRST_BUILD") > 200) {
            // Normal - new builds expected
            logger.info("High first-build miss rate: {} new artifacts", 
                       missByReason.get("FIRST_BUILD"));
        }
    }
    
    // Adaptive cache sizing based on workload patterns
    private void adaptCacheSizing(CacheMetrics metrics) {
        // Adjust hot cache size based on access patterns
        double accessSkewness = metrics.getAccessSkewness();
        long currentHotCacheSize = metrics.getHotCacheSize();
        
        if (accessSkewness > 0.8) { // Very skewed access pattern
            // Reduce hot cache size - most artifacts are rarely accessed
            long newSize = (long) (currentHotCacheSize * 0.9);
            hotCacheManager.resizeTo(newSize);
            
        } else if (accessSkewness < 0.4) { // More uniform access
            // Increase hot cache size - more artifacts need fast access
            long newSize = (long) (currentHotCacheSize * 1.1);
            hotCacheManager.resizeTo(newSize);
        }
    }
}
